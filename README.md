# Hand to Hand

**Hand-to-Hand**는 청각장애인을 위한 수서생성 서비스 입니다.





## Installation

---------------
To install package
```bash
    $ git clone https://github.com/Tobigs-team/hand-to-hand.git
    $ cd hand-to-hand
```
